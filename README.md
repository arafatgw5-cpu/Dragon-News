# 📰 Dragon News

A modern news portal built with **Next.js**, **Tailwind CSS**, and **Better Auth**. Users can browse category-based news, view details, and login/register with email or social authentication.

## 🔗 Live Link

https://dragon-news-blush.vercel.app

## 🚀 Features

- 📰 Category based news browsing
- 📄 News details page
- 🔐 Email & password authentication
- 🌐 Google / GitHub social login
- 👤 Dynamic navbar with logged-in user info
- 📱 Fully responsive design
- ⚡ Fast performance with Next.js App Router

## 🛠️ Technologies Used

- Next.js
- React
- Tailwind CSS
- Better Auth
- MongoDB
- React Hook Form
- Lucide React
- Vercel
